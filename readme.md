# ▶️ Cómo Ejecutar el Proyecto

## 1️⃣ Instalar Java

Descargar e instalar el JDK desde:
https://www.oracle.com/java/technologies/downloads/

## 2️⃣ Clonar el repositorio

```bash
git clone https://github.com/usuario/proyecto-java.git
```

## 3️⃣ Ingresar al proyecto

```bash
cd proyecto-java
```

## 4️⃣ Compilar el programa

```bash
javac Main.java
```

## 5️⃣ Ejecutar el programa

```bash
java Main
```
